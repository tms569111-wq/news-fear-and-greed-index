# 제외한 업로드 파일

이 패키지에는 다음 파일을 일부러 포함하지 않았습니다.

1. `비번.txt`
   - 비밀번호/민감정보로 보이는 내용이 있어 제외했습니다.

2. 개인정보가 들어간 원본 UGRP 보고서 PDF/HWP
   - 이름, 학번, 이메일, 전화번호 등이 포함될 수 있어 제외했습니다.
   - 대신 `legacy/report/졸업프로젝트_블라인드버전.pdf`만 포함했습니다.

3. HWP 양식 원본
   - Codex 구현에는 필요하지 않아 제외했습니다.

필요하면 본인 로컬에서 별도 백업만 하고, GitHub나 Codex 작업공간에는 올리지 않는 것을 권장합니다.
